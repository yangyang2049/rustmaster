Rust 从入门到放弃是一款专注于 Rust 编程语言学习的移动应用，提供系统化的课程教程、语法速查、知识测试和面试题库，帮助开发者从零基础到熟练掌握 Rust 编程。应用包含 12 个课程单元，涵盖 Rust 基础、所有权系统、复合数据类型、集合类型、错误处理、模块系统、泛型与 Trait、智能指针、并发编程、异步编程、高级特性和实战项目等核心主题。每个章节都配有详细的讲解、示例代码和知识点总结，支持 Markdown 渲染和代码复制功能。语法速查功能提供超过 200 个语法条目，方便快速查找和参考。知识测试功能帮助巩固学习成果，面试题库包含 100 道精选面试题，涵盖从基础到高级的各个主题，每题包含核心要点、常见错误回答、代码推理和追问方向。应用界面简洁清晰，支持深色模式，提供朗读和分享功能，是 Rust 学习者的理想选择。
